Despliega tu Proyecto Web

Paso 1: Organiza tus archivos

Antes de iniciar, asegúrate de tener esta estructura:


  • /MiProyectoWeb
  • ├── index.html
  • ├── EstilosBH.css
  • ├── Elpo.js
  • ├── /Imgs — logos, banners e íconos
  • ├── /PDF's — documentos del curso
  • └── /Banner — imágenes del carrusel

Usa nombres sin tildes ni espacios para evitar errores al enlazar.

Paso 2: El corazón visual | El CSS

CSS

Controla colores, fuentes, espaciado y layout de tu sitio.

Enlace correcto

<link rel="stylesheet" href="Estilos.css" />

Diseño responsivo

Visualiza tu diseño tal como lo imaginaste, en cualquier dispositivo.

Paso 3: Dale vida | JavaScript

Incluye tu script justo antes de </body> para mejor rendimiento.

Paso 4: Abre tu página & Paso 5: Prueba como visitante

Abrir en navegador


Haz doble clic en index.html y revisa todo el funcionamiento.

Checklist de pruebas


  • ¿Se ven las imágenes?
  • ¿Abren los PDF's?
  • ¿Funciona el login?
  • ¿Se activa el scrollspy?

Paso 6: Sube tu página a la nube

Comparte tu proyecto por Drive, GitHub Pages o Netlify. Asegúrate que los enlaces funcionen.


Publicar en GitHub Pages

Conclusión

Montar una web va más allá del código: se trata de orden, conexión precisa y pruebas constantes.
